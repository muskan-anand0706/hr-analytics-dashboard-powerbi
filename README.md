# HR Analytics Dashboard | Power BI

**Brief One Line Summary:**  
Interactive dashboard to analyze employee attrition and workforce distribution.

**Overview:**  
Built a Power BI dashboard using employee data (1,000+ records) to visualize workforce patterns and attrition trends across departments, roles, gender, and marital status.

**Problem Statement:**  
Organizations often struggle to identify factors causing employee attrition. This project aims to uncover trends and insights to support HR decision-making and retention strategies.

**Dataset:**  
Publicly available HR dataset (raw and cleaned data files included).

**Tools and Technologies:**  
Power BI | Data Cleaning | Data Visualization

**Methods:**  
- Data cleaning and preprocessing  
- Visualization of workforce distribution across departments, roles, gender, and marital status  
- Interactive dashboard with slicers and filters to explore different segments

**Key Insights:**  
- Attrition rates vary across departments and roles  
- Workforce distribution differs by gender and marital status  
- HR can use these insights for planning and retention strategies

**Dashboard / Model / Output:**  
Interactive Power BI dashboard with:  
- Filters for gender and marital status  
- Visualizations of department-wise and role-wise workforce distribution  
- Key metrics like total employee count and attrition rate

**How to Run this Project:**
1.	Open the provided Power BI file (.pbix)
2.	Explore the dashboard using interactive slicers for gender and marital status
3.	Review the insights and visualizations

**Results & Conclusion:**  
- Clear visibility into factors affecting attrition  
- Insights support workforce planning and retention decisions  
- Dashboard provides an easy-to-understand overview of HR metrics

**Future Work:**  
- Add predictive analytics to forecast potential attrition  
- Include sentiment analysis from employee feedback  
- Create drill-through reports for deeper role-specific insights
